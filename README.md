# MiniStore

MiniStore is a small e-commerce frontend project made with React JS.
In this project, users can see products and add them to the cart.

This project is only for learning React and frontend development.

---

## What this project does

- Shows products from an API
- Allows users to add products to cart
- Increase or decrease product quantity
- Remove product from cart
- Shows total price
- Shows a thank you page after checkout

---

## Tools used

- React JS
- React Router
- Tailwind CSS
- Axios



---

## Why I made this project

I made this project to practice:
- React components
- useState and useEffect
- API data fetching
- Cart logic
- Basic routing

---

## Notes

- This project has no backend
- No login or payment system
- API is used only to get product data
